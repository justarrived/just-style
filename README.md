# Just Style
Based on [Sassy Starter][0]. 

## Install

Make sure you have the Gulp CLI installed. If you don't, run:
```
npm i -g gulp
```

Install dependencies:
```
$ npm i
```

## Contribute

Follow the [airbnb styleguide][1] for writing SCSS.

Run:
```
gulp
```
Starts server at `localhost:3000`

## Build
```
gulp build
```
Outputs to `dist/`
